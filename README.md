# ⚽ FIFA World Cup 2026 Player Performance Analysis

## 📌 Project Overview

This project analyzes FIFA World Cup 2026 player performance data using Python and Power BI. The objective is to clean, explore, and visualize player statistics to uncover meaningful insights into player performance, team contributions, and tournament trends.

The project demonstrates the complete Data Analytics workflow, including data preprocessing, exploratory data analysis (EDA), and dashboard creation.

---

## 🎯 Objectives

- Perform data cleaning and preprocessing
- Analyze player performance statistics
- Identify top-performing players
- Compare teams based on performance metrics
- Create interactive Power BI dashboards
- Generate business insights from football data
