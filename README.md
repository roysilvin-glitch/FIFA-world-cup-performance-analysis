# ⚽ FIFA World Cup 2026 Player Performance Analysis

## 📌 Overview
This project analyzes FIFA World Cup 2026 player performance data using **Python** and **Power BI**. It includes data cleaning, exploratory data analysis (EDA), and an interactive dashboard to visualize player and team performance.

## 🚀 Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Interactive Power BI Dashboard
- Player & Team Performance Analysis
- Key Performance Insights

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Power BI
- Jupyter Notebook

## 📊 Key Insights
- Top-performing players
- Team performance comparison
- Goal and assist analysis
- Player rating distribution
- Interactive dashboard with filters

## 📂 Files
- `fifa_world_cup_2026_player_performance.csv` – Dataset
- `fifa_world_cup_analysis.ipynb` – Python analysis
- `fifa_analysis_dashboard.pbix` – Power BI dashboard
